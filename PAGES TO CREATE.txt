### 6.2 Create Profile in SQL

Run this SQL to create the admin profile:

```sql
INSERT INTO profiles (id, full_name, email, role)
VALUES (
  (SELECT id FROM auth.users WHERE email = 'admin@school.edu'),
  'Admin User',
  'admin@school.edu',
  'admin'
);
password: admin123
```
